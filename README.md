# JavaScript_Projects

A collection of interactive web applications built using HTML, CSS, and Vanilla JavaScript. This repository serves as a playground for practicing DOM manipulation, canvas rendering, and logic implementation.

---

## 🎮 Projects Included

### 1. Split Bill App
A practical utility tool to help users calculate how to divide a bill among multiple people, including tip percentages.
* **Tech Stack:** HTML, CSS, JavaScript.
* **Features:** Calculates tip amounts and per-person totals in ₹ (Rupees).
